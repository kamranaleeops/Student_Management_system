# 🎓 Student Management System

A simple and efficient **Student Management System** built with **Python**.  
This project allows users to manage student records using a **command-line interface (CLI)** and stores data in a **JSON file** for persistence.

---

## 🚀 Features

- ➕ Add new students
- 📋 View all students
- 🔍 Search student by roll number
- ✏️ Update student information
- ❌ Delete student records
- 🔤 Sort students by name
- 📊 Count total number of students
- 💾 Data stored using JSON file

---

## 🛠️ Technologies Used

- Python
- JSON
- Command Line Interface (CLI)
